# 🌐 Personal Portfolio

My personal portfolio website built with **Next.js** and **Tailwind CSS**.
Clean, minimal, and functional.

🔗 **Live Site:** [https://github.com/martinsharma18](https://github.com/martinsharma18)

## 🚀 Built With

* **Next.js** – React framework for production
* **Tailwind CSS** – Utility-first CSS framework
* **Resend** – Email delivery with secure contact forms
* **TypeScript** – Type safety (because I like my code not breaking)

This portfolio showcases:

* Skills and tech stack

* Ways to get in touch
